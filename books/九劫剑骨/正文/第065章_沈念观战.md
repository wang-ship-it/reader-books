# 第065章 沈念观战

第二日，丹峰弟子也来了擂台。

沈念站在人群后方，穿着丹峰青白色弟子服，手里攥着一只小药瓶。她个子不高，被前面的人挡住，只能踮脚看擂台。苏小蛮发现后，把她拉到一块石阶上。

“站这儿，看得清。”苏小蛮道。

沈念小声道谢，视线却一直落在沈烬身上。

沈烬正在台下候战。右肩的药布藏在衣服里，左拳也缠着薄纱。别人只看见他连赢、排名上涨、废脉翻身，沈念却能看见他抬手时细微的停顿，看见他每次呼吸都比平时慢一点。

她从小就会看哥哥疼不疼。

以前在沈家，她只会害怕。怕自己拖累他，怕族老迁怒，怕沈青阳带人来找麻烦。现在她仍怕，却不再只想哭。

丹峰师姐林芷站在旁边，道：“你若总怕他伤，就学不好医。”

沈念怔了怔。

林芷看着擂台：“医修不是让人不流血。修行路上没人能不流血。你要学的是，他流血时，你知道先按哪里，先救哪条脉，先保哪口气。”

沈念握紧药瓶，点了点头。

台下，沈烬似有所感，回头看了过来。

兄妹隔着人群对视。

沈念原本想喊“小心”，话到嘴边又停住。苏小蛮说过，剑修上台前最怕别人喊小心，因为那会让他们想起身后的人，出剑就会软一分。

于是她没有喊。

她只是用力点头。

沈烬看见了。

胸口那根绷了许久的弦，忽然松了一点。不是松到懈怠，而是他知道自己不是一个人在往前走。沈念在丹峰学药，楚狂歌在台上拼刀，苏小蛮用她的方式守账也守人，江雪衣和秦夜在规矩里记证。

他不能倒。

但也不必把所有东西都背得像孤身赴死。

陆照渊在断剑里道：“看够了没？再看你妹妹就要以为你怕了。”

沈烬转回身：“我不怕。”

“怕也正常。”陆照渊道，“不怕的人，要么蠢，要么死得早。怕了还知道往前走，才勉强像个剑修。”

第二轮名册很快贴出。

沈烬对叶坤。

叶坤站在另一侧，玄铁短枪靠在肩上，先朝赵无极所在方向行了一礼。那动作不明显，却被沈烬、楚狂歌和苏小蛮同时看见。

沈念也看见了。

她不知道赵无极和沈烬之间所有暗线，却能从那一礼里看出恶意。她低头打开药瓶，确认里面的止血散还在。

林芷轻声道：“别慌。”

沈念点头：“我不慌。”

她说得很轻，却是真的。

擂台钟响，沈烬走上台。沈念站在石阶上，手指紧紧攥着药瓶，眼睛却没有躲开。

这一次，她要看完整场。

擂台风从下方吹来，带着汗味、药味和石面被兵器刮出的尘味。沈念以前不喜欢这些味道，觉得它们都和受伤有关。可现在她强迫自己记住，因为以后哥哥每一次从擂台下来，她都要从这些味道里分辨毒、血、骨热和普通擦伤。

林芷看见她的手终于不抖，轻轻点了点头。丹峰弟子不必上擂台，但真正好的医修，心不能比上擂台的人更软。沈念看着沈烬走向叶坤，忽然很清楚地知道，自己也该开始修自己的道了。

她把小药瓶贴在掌心，开始在心里默背林芷教过的伤势顺序：先止喷涌之血，再护心肺，再查骨裂，最后才问疼不疼。以前她总想第一句就问哥哥疼不疼，如今才明白，那句话救不了人。她要学会在别人都慌乱的时候，把手按在最该按的位置。

叶坤向赵无极行礼的那一瞬，林芷也看见了。她没有多说，只把一枚丹峰内门的银针递给沈念，让她藏进袖中。“若有人在台下动药，先扎自己指尖，别让毒气顺呼吸走。”这句话说得很轻，却让沈念背后泛起凉意。原来擂台并不只是台上两个人的战斗，台下的药、眼神、名册、执事站位，全都可能变成刀。她抬头看着哥哥的背影，第一次觉得自己必须快一点长大。

沈念忽然想起离开沈家那天，沈烬也是这样站在她前面，把所有人的目光挡住。那时候她只会躲在他身后哭，现在她不想再躲。她把银针藏好，又把止血散、护脉丹、稳骨符在袖袋里按顺序排了一遍。等沈烬下台时，无论他是赢是输，她都要第一时间知道该拿哪一样。

林芷没有再教她新的药理，只让她看。看叶坤起枪前的肩，看沈烬避枪时右脚落点，看台下赵无极身边弟子的表情。医修若只看伤口，就永远慢一步；真正的救人，要在伤还没落下前，先知道它会落在哪里。
